🚫 Smart Adblock

Advanced filtering using AdGuard, ABPVN, and 1Hosts. Synchronized hourly via GitHub Actions.

🌍 ECS Injection

EDNS Client Subnet (RFC 7871) ensures geo-accurate resolution for Akamai, Gcore, and global CDNs.

🔄 CNAME Injection

Native redirection engine. Force specific CDN clusters or override domain resolution at the Edge.

🛡️ Geo-Bypass

Sequential failover with detection for blocked results (127.0.0.1) and automatic fallback to Mullvad DNS.

⚡ Edge Optimization

ANY, AAAA, PTR, and HTTPS query types are filtered at the Edge to maximize speed and efficiency.

💎 Built for Scale

Optimized for 100,000 requests/day. Supports 10-20 devices per account comfortably and for free.
